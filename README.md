# Unit Converter (Vanilla JS)

A simple unit converter for distance and temperature with instant results.

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Run Locally
Open `src/index.html` in your browser.

## Project Structure
- `src/index.html` — UI
- `src/styles.css` — styling
- `src/app.js` — logic

## What I practiced
- DOM events and input parsing
- Pure conversion functions
- Presenting results with small formatting helpers
